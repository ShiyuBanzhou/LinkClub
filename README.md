# 零刻软研社前端工程师学习指南

> 注：本手册假设读者只有 C、python 等一门语言基础，期望在完成本手册学习任务后，可以授入正式项目开发。

想成为一名合格的软件工程师，多个领域都需要掌握一定知识：操作系统、计算机网络、编码语言、算法、前端、后端等。同时大家也不用担心，如果你是计算机相关专业的同学，操作系统、计算机网络、算法都是在你升入高年级后老师会教授的课程，现在学习这些知识也可以使你后续的学习过程更加轻松。

但是只学理论知识，没有实践，无法衡量自己的掌握程度，最好是能有看得见成果用以衡量当前水乎所以在手册中途会加入实际项目中的场景进行学习。

在每个阶段，都要结合着基础知识和实践场景，相互印证。

这里推荐大家学习过程中给自己拟定学习计划，以合理安排社团时间以及学校学习时间

TypeScript
在我们的生产项目中，使用 TS 大概占 90%，ES6 占 10%。
TS 可以理解为“ESn + 类型系统”。因此学完 ES 就已经学会了 80%的 TS，再了解一下 TS 的类型系统（比如 enum、interface……）就可以愉快地玩耍了。
有其他编程语言开发经验，基础比较好的同学也可以跳过 ES，直接从 TS 开始学习。学习 TS
[TS 教程 1](https://www.typescriptlang.org/docs/handbook/typescript-in-5-minutes.html)
[TS 教程 2](https://www.typescriptlang.org/docs/handbook/intro.html)

可以利用 TS Playground 测试 TS 语法
